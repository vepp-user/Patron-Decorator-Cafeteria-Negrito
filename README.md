# Patron-Decorator-Cafeteria-Negrito
Descripción

Proyecto desarrollado en Java aplicando el patrón de diseño Decorator para gestionar bebidas y complementos de la cafetería El Negrito.

Parte I

Implementación del patrón Decorator para:

Expreso
Batido
Descafeinado
Tostado Negro

Complementos:

Leche
Moca
Crema
Soya

Parte II

Se modificó el diagrama UML

Se incorporaron tres tamaños:

Normal (N)
Mediano (M)
Grande (G)

Los precios de bebidas y complementos varían según el tamaño seleccionado.

Tecnologías
Java
NetBeans
GitHub
Patrón utilizado

Decorator
